# Python Fundamentals: Core Logic & Conditional Control Flow

Welcome to my Python learning repository! This repository serves as a structured documentation of my journey through foundational Python programming, data validation, and complex conditional decision-making algorithms. 

The focus of this phase (Chapters 1 to 5) is mastering clean syntax, input sanitization, and transitioning from nested structures to efficient, linear programmatic logic.

---

## 🚀 Key Concepts Mastered

* **Dynamic Typing & Type Casting:** Handling user inputs safely using explicit casting (`int()`, `float()`, `str()`).
* **String Manipulation & Engineering:** Utilizing string slicing, case-insensitivity tactics (`.lower()`), and native substring containment tracking (`in`, `not in`).
* **Mathematical & Structural Operators:** Implementing floor division (`//`) and modulus (`%`) for algorithmic reductions.
* **Advanced Control Flow:** Constructing multi-branch conditional statements (`if-elif-else`) with optimized logical operators (`and`, `or`) to prevent "Pyramid of Doom" deep-nesting errors.
* **Boundary Input Validation:** Implementing logical filters to screen, sanitize, and validate user inputs before runtime execution.
